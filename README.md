# AI Topper Smart 🔥

منصة تعليمية ذكية تجريبية (Prototype) لطالب الصف التاسع والثالث الثانوي في الجمهورية اليمنية، مبنية بالكامل بـ HTML/CSS/JS بدون أي إطار عمل خارجي، وتعمل من رابط واحد عبر GitHub Pages.

🔗 **الموقع المباشر:** https://wkkwkww411-ai.github.io/ai-topper-smart/

## الأنظمة المتوفرة

| النظام | الوصف |
|---|---|
| 📊 لوحة التحكم | إحصائيات الطالب، سلسلة الدراسة، الترتيب، رسم بياني للدرجات |
| 🤖 المعلم الذكي | شات تفاعلي بأسلوب سقراطي، بثلاث شخصيات، ويعمل بذكاء اصطناعي حقيقي داخل المتصفح (WebLLM / DeepSeek) |
| 📖 الكراسة الرقمية | عارض دروس مع تلخيص وشرح بالذكاء الاصطناعي، ونظام ملاحظات |
| 📝 الاختبارات | اختبارات تفاعلية فعلية مع تصحيح فوري، وعد تنازلي للاختبار الوزاري |
| 📷 حل المسائل بالتصوير | نموذج أولي لتحليل صور المسائل |
| 🎮 وضع التحدي | نقاط، شارات، لوحة صدارة، مهمة يومية |
| 👨‍🏫 حجز المدرسين | حجز حصص خصوصية تفاعلي |
| 💬 مجتمع الطلاب | منشورات ونقاشات بين الطلاب |
| ⏰ إدارة الوقت | جدول يومي وتحليل الوقت الضائع |
| 👨‍👩‍👦 بوابة ولي الأمر | تقارير أسبوعية وتنبيهات مبكرة |
| 🗺️ خريطة النجاح | تايم لاين لمسار الطالب نحو أوائل الجمهورية |
| 🏅 الإنجازات | معرض شارات |
| 👤 الملف الشخصي | ملخص بياني لجميع الدرجات |

## التقنيات

- HTML / CSS / JavaScript خام (بدون React أو أي build step)
- [Chart.js](https://www.chartjs.org/) للرسوم البيانية
- [WebLLM](https://github.com/mlc-ai/web-llm) لتشغيل نموذج ذكاء اصطناعي (DeepSeek-R1-Distill) محلياً داخل متصفح الطالب، بدون أي خادم أو مفتاح API
- خطوط Cairo و Tajawal من Google Fonts

## الأمان

هذا المشروع **لا يحتوي على أي مفاتيح API مكشوفة**. أي ربط مستقبلي بخدمات مدفوعة (مثل Groq أو Anthropic) يجب أن يمر عبر خادم وسيط (مثل Cloudflare Workers) يُبقي المفتاح سرياً — لا تضع أي مفتاح داخل هذا المستودع مباشرة.

## قاعدة البيانات (قيد الإعداد)

راجع [`firebase-integration-guide.md`](./firebase-integration-guide.md) لخطوات ربط المشروع بتسجيل دخول جوجل وقاعدة بيانات Firestore لكل طالب.

## التشغيل محلياً

فقط افتح `index.html` في أي متصفح — لا يحتاج أي تثبيت أو بناء.
